# Publication Plan

## Target Venues
- NeurIPS / ICLR (AI/ML Track)
- Nature Geoscience (Applications Track)

## Working Title
*Continuous-Lag Causal Discovery and Representation Learning for Real-Time Seismic Forecasting*

## Abstract Structure
1. **Background:** Unpredictability of earthquakes and the challenge of false positives in early-warning systems.
2. **Method:** Introduction of EPGNN, a Multimodal Graph Neural Network treating sensor arrays as interconnected spatio-temporal nodes.
3. **Results:** Demonstration of >90% predictive accuracy and superior F1-Scores compared to classical threshold-based seismology methods using the STEAD dataset.
